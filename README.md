### Tailwind CSS Utility Classes (Notes)
PS - USE TAILWIND PLAY
- `text-lg` → sets font-size **and** line-height
- `text-blue-500` → text color (blue shade)
- `text-center` → text alignment
- `mt-2` → margin-top (0.5rem)
- bg-sky-200
- w-full -> full width
- border-violet-600
- rounded-md
- p-2 padding 2
- my-2 -> margin in y axis both top and bottom 
- mx-2 margin left and right both along x axis
- font-mono, font-extrabold, 
- text-[30px] 


### Notes
- Tailwind CSS uses **utility-first classes**
- Styles are **generated on the fly** based on usage
- No unused CSS is shipped to production
- JIT compiler -> only includes the styes that you use to keep your final CSS small just in time compiler
- mobile-first responsive -> by default uses min-width md:block hidden medium sized display:block
